2048-Clone

My 2048 clone is my own version of the mobile game 2048, in this game you have to stack as many equal blocks as possible by pushing them into each direction, after each move a new block will always appear in a random place. In my version you can choose between different game sizes, for each size there is a separate highscore which is stored in a SQL database. This highscore input and output is realized by a Node.js server. The game also has an audio player for background music and a light and dark mode.

Pokedex

On the website you can then see all Pokemons on a color-coded map. If you click on this card, you will see all the values of the respective Pokemon link to website.

Miningmark Industry

Miningmark-Industry is my first self-programmed browser game. It is an Idle-Clicker Game, which runs in the browser without a backend. The game runs exclusively with HTML, CSS, JavaScript for the design I used Bootswatch. Cockies are used to save the game progress, this happens automatically, but you can also export your game progress to a txt file and import it again. You can also choose between different display options in the menu, e.g. dark or light mode and a few others. Unfortunately this game is only available in German link to website
